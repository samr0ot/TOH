# Tower of Hanoi Using Recursion in C

## Description
This project implements the Tower of Hanoi problem using recursion and file handling.

## Features
- Recursive algorithm
- Asks user for number of disks
- File logging of moves
- Writes down the minimum moves required to solve the problem

## Compile & Run
gcc project2.c
./a

## Formula
Total moves = 2^n - 1
